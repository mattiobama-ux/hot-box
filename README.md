# 🔥 Hot Box Website

A modern, responsive website built using HTML, CSS, and JavaScript. This project is designed to provide a clean user experience with a fast-loading interface and attractive design.

## 🚀 Features

- Responsive design for mobile, tablet, and desktop
- Modern and clean UI
- Fast loading speed
- Easy navigation
- Contact section
- Smooth animations
- SEO-friendly structure

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
